Vue：是一套用于构建用户界面的渐进式框架；Vue被设计为可以自底向上逐层应用，Vue的核心库只关注视图层，不仅容易上手，还便于与的第三方库或既有项目整合；而且，党羽现代化工具链以及各种支持列库结合使用时，Vue也完全能够为负责的单页应用提供驱动。

特点：
	简洁： HTML 模板 + JSON 数据，再创建一个 Vue 实例，就这么简单。
	数据驱动： 自动追踪依赖的模板表达式和计算属性。
	组件化： 用解耦、可复用的组件来构造界面。
	轻量： ~24kb min+gzip，无依赖。
	快速： 精确有效的异步批量 DOM 更新。
	模块友好： 通过 NPM 或 Bower 安装，无缝融入你的工作流
	
目标：通过尽可能简单的API实现响应的数据绑定和组合的视图组件
